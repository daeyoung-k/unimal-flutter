## 2026-07-24 팔로우 가능한 개인 큐레이션 지도 ⭐ 유력

> "지도 = 콘텐츠 채널". 우리지도/비밀지도 인프라의 자연스러운 확장.

### 컨셉

- 폐쇄형 지도를 그룹 없이 **나 혼자 여러 개** 만들 수 있다 (예: 캠핑 지도, 카페 지도).
- 혼자 간직하며 채워나가다가, **공유 설정**하면 다른 유저가 그 지도를 **팔로우** 가능.
- 지도 주인만 게시글을 남길 수 있음. 팔로워는 **좋아요/댓글까지만** — 이 제한이 큐레이션 품질을 지킴.
- 주인이 새 기록을 남기면 팔로워에게 **앱푸시 알림**.

### 왜 좋은가

- 크리에이터 루프: 팔로워 → 주인은 채울 동기, 팔로워는 푸시로 복귀. 현재 스토맵에 없는 리텐션 고리.
- 스토맵 정체성("내 기록이 지도가 된다")과 정확히 일치.
- BM 씨앗: 나중에 지도 구독, 프리미엄 지도로 확장 가능.

### 설계 시 미리 정할 것

- [ ] 게시글이 여러 지도에 속할 수 있는가? (데이터 모델에 큰 영향)
- [ ] 공개 범위 3단계: 비공개 / 링크 공유 / 전체공개
- [ ] 푸시 빈도 제어 — 하루 여러 개 올리면 묶어서 1회
- [ ] 좋아요/댓글 단위: 게시글 단위 vs 지도 단위
- [ ] 팔로우 지도 탐색 진입점 (검색? 추천? 프로필?)

### 순서

우리지도 인프라 → 개인 지도 다중 생성 → 공유/팔로우 → 팔로워 푸시

## 2026-07-24 멤버 모집 / 파티원 모집 🧊 보류

특정 마커에 "같이 갈 사람?" — 제한 인원 모집, 신청서 받고 승인/거절. 게임 파티 모집 느낌 (예: 이 시간에 런닝할 사람).

**보류 이유**: 기능이 아니라 다른 제품에 가까움 (당근 모임/문토 영역). 신청·승인, 노쇼, 신고·차단, 대면 안전 이슈 등 운영 부담이 1인 개발 범위 초과. 기록/발견 정체성 희석 위험.

**부활 조건**: 팔로우 지도가 자리 잡은 뒤 "지도 주인이 여는 이벤트" 형태로 가볍게 재검토.

## 2026-07-24 기타 후보 (브레인스토밍)

- **회고 리마인드**: "1년 전 오늘 이 장소의 기록" 푸시. 구현 비용 낮음(푸시 인프라 있음), 리텐션 직접 효과. 우선순위 높음.
- **내 지도 스냅샷 공유**: 내 지도를 이미지로 내보내 인스타 공유. 마케팅 비용 0원 성장 채널.
- **지도 채우기 시각화**: 기록 남긴 지역이 밝아지는 탐험 게이지.
- **월말 기록 요약 카드**: "이번 달 7곳에 기록" 공유 카드.
- **장소 히스토리**: 같은 장소에 쌓인 여러 사람의 기록을 시간순으로 ("이 카페의 이야기들").
- **테마 지도 큐레이션**: 시즌성 수동 큐레이션 (벚꽃, 심야 맛집).
- **수익화(후순위)**: 카드 스트립 네이티브 광고, 우리지도 개수 제한 → 구독 해제, 지도 커스텀 테마.
