# 지도 마커 재조회 정책 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 지도 공간 이동, 30초 데이터 freshness, 텍스트 마커 표현 전환을 분리해 불필요한 API 호출을 줄이면서 활성 지도에는 새 게시글을 주기적으로 반영한다.

**Architecture:** 카메라 재조회 판단을 순수 `MapReloadPolicy`로 분리하고, 메인 지도는 정책이 반환한 이유에 따라 기존 증분 로더를 호출한다. API 실패는 `null`, 정상 0건은 빈 목록으로 구분하며, 강제 새로고침도 응답 성공 후에만 기존 오버레이를 교체한다. 마지막 성공 시각 기준 one-shot 타이머가 30초 freshness를 담당한다.

**Tech Stack:** Flutter, Dart 3.8, `flutter_naver_map 1.4.4`, GetX, `geolocator`, `http`, `flutter_test`

## Global Constraints

- 점·말풍선 표현 변경은 API 재조회 조건에 포함하지 않는다.
- 공간 이동 임계값은 화면 짧은 변의 30%에 해당하는 미터 거리다.
- API 줌 변경은 `_apiZoomFor(currentZoom) != _lastQueriedApiZoom`으로 판단한다.
- 카메라 idle 디바운스는 300ms다.
- freshness 간격은 마지막 성공 응답 기준 30초, 실패 재시도는 15초다.
- 앱이 resumed이고 지도 탭이 활성화됐으며 카드·스택이 닫힌 경우에만 자동 조회한다.
- HTTP 실패와 파싱 실패는 기존 마커를 유지하고, 정상 빈 목록만 기존 마커를 제거한다.
- `refreshMap()`은 응답 성공 전 기존 오버레이와 캐시를 삭제하지 않는다.
- 백엔드, 내 지도, WebSocket/SSE, 텍스트 크로스페이드는 이번 계획에서 변경하지 않는다.
- 기존 사용자 변경인 `.vscode/launch.json`과 `lib/state/auth_state.dart`는 수정하거나 포함하지 않는다.
- 작업 단위별 커밋을 만들지 않는다. 전체 구현과 검증 후 변경 내용을 요약하고 사용자 승인 뒤에만 커밋한다.

```dart
import 'dart:ui';

enum MapReloadReason {
  none,
  initial,
  apiZoomChanged,
  viewportMoved,
  stale,
  appResumed,
  manual,
}

class MapReloadPolicy {
  const MapReloadPolicy._();

  static const double viewportMovementFraction = 0.30;
  static const Duration freshnessInterval = Duration(seconds: 30);
  static const Duration failureRetryInterval = Duration(seconds: 15);
  static const Duration cameraDebounce = Duration(milliseconds: 300);

  static double movementThresholdMeters({
    required Size viewportSize,
    required double meterPerDp,
  }) {
    final shortestSide = viewportSize.shortestSide;
    return shortestSide * meterPerDp * viewportMovementFraction;
  }

  static MapReloadReason spatialReason({
    required bool initialized,
    required int currentApiZoom,
    required int lastApiZoom,
    required double movedMeters,
    required double movementThresholdMeters,
  }) {
    if (!initialized) return MapReloadReason.initial;
    if (currentApiZoom != lastApiZoom) {
      return MapReloadReason.apiZoomChanged;
    }
    if (movedMeters >= movementThresholdMeters) {
      return MapReloadReason.viewportMoved;
    }
    return MapReloadReason.none;
  }

  static bool isFreshnessDue({
    required DateTime? lastSuccessfulAt,
    required DateTime now,
  }) {
    if (lastSuccessfulAt == null) return true;
    return now.difference(lastSuccessfulAt) >= freshnessInterval;
  }

  static bool canAutoRefresh({
    required bool isAppResumed,
    required bool isMapTabActive,
    required bool isInteractionOpen,
    required bool isLoading,
  }) =>
      isAppResumed &&
      isMapTabActive &&
      !isInteractionOpen &&
      !isLoading;
}
```

List<MapPost>? decodeMapPostsResponse(http.Response response) {
  if (response.statusCode != 200) return null;
  try {
    final body = jsonDecode(utf8.decode(response.bodyBytes));
    final data = body['data'];
    if (data is! List) return null;
    return data
        .map((e) => MapPost.fromJson(e as Map<String, dynamic>))
        .toList();
  } catch (_) {
    return null;
  }
}

Future<List<MapPost>?> getMapLocationPosts({
  required double latitude,
  required double longitude,
  int zoom = 14,
}) async {
  try {
    final url = ApiUri.resolve('board/map/post', {
      'latitude': latitude.toString(),
      'longitude': longitude.toString(),
      'zoom': zoom.toString(),
    });
    final headers = await _authHeaders();
    final response = await ApiClient.get(url, headers);
    final posts = decodeMapPostsResponse(response);
    if (posts == null) {
      _logger.e('지도 마커 조회 실패: ${response.statusCode}');
    }
    return posts;
  } catch (e, st) {
    _logger.e('지도 마커 조회 예외', error: e, stackTrace: st);
    return null;
  }
}

  viewportSize: viewportSize,

        _lastQueriedTarget!.latitude,

        currentTarget.latitude,
        currentTarget.longitude,

  initialized: _mapInitialized,
  currentApiZoom: currentApiZoom,
  lastApiZoom: _lastQueriedApiZoom,
  movedMeters: movedMeters,
  movementThresholdMeters: movementThreshold,
);

if (reason == MapReloadReason.initial) {
  _mapInitialized = true;
  return;
}
if (reason == MapReloadReason.none) {
  if (kDebugMode) {
    debugPrint('[map] reload skipped reason=withinViewport '
        'moved=${movedMeters.toStringAsFixed(0)}m '
        'threshold=${movementThreshold.toStringAsFixed(0)}m');
  }
  return;
}

_cameraDebounce?.cancel();
if (kDebugMode) {
  debugPrint('[map] reload scheduled reason=${reason.name} '
      'apiZoom=$currentApiZoom debounce=300ms');
}
_cameraDebounce = Timer(MapReloadPolicy.cameraDebounce, () {
  _loadMapMarkers(
    currentTarget.latitude,
    currentTarget.longitude,
    currentApiZoom,
    rawZoom: currentZoom,
    reason: reason,
  );
});

Add the following methods to the map state:

```dart
void _scheduleFreshness({Duration? delay}) {
  _freshnessTimer?.cancel();
  _freshnessTimer = Timer(
    delay ?? MapReloadPolicy.freshnessInterval,
    _onFreshnessDue,
  );
}

void _onFreshnessDue() {
  if (!MapReloadPolicy.canAutoRefresh(
    isAppResumed: _appLifecycleState == AppLifecycleState.resumed,
    isMapTabActive: _isMapTabActive,
    isInteractionOpen: _isMapInteractionOpen,
    isLoading: _isLoadingMarkers,
  )) {
    _freshnessRefreshPending = true;
    _pendingFreshnessReason = MapReloadReason.stale;
    if (kDebugMode) {
      debugPrint('[map] freshness deferred reason=autoRefreshBlocked');
    }
    return;
  }
  unawaited(_reloadCurrentCamera(MapReloadReason.stale));
}

Future<void> _consumePendingFreshness() async {
  if (!_freshnessRefreshPending) return;
  if (!MapReloadPolicy.canAutoRefresh(
    isAppResumed: _appLifecycleState == AppLifecycleState.resumed,
    isMapTabActive: _isMapTabActive,
    isInteractionOpen: _isMapInteractionOpen,
    isLoading: _isLoadingMarkers,
  )) return;
  final reason = _pendingFreshnessReason;
  _freshnessRefreshPending = false;
  await _reloadCurrentCamera(reason);
}

Future<void> _reloadCurrentCamera(MapReloadReason reason) async {
  final controller = _mapController;
  if (controller == null) return;
  final camera = controller.nowCameraPosition;
  await _loadMapMarkers(
    camera.target.latitude,
    camera.target.longitude,
    _apiZoomFor(camera.zoom),
    rawZoom: camera.zoom,
    reason: reason,
  );
}

@override
void didChangeAppLifecycleState(AppLifecycleState state) {
  _appLifecycleState = state;
  if (state != AppLifecycleState.resumed) return;
  final due = MapReloadPolicy.isFreshnessDue(
    lastSuccessfulAt: _lastSuccessfulMapRefreshAt,
    now: DateTime.now(),
  );
  if (due) {
    _freshnessRefreshPending = true;
    _pendingFreshnessReason = MapReloadReason.appResumed;
    unawaited(_consumePendingFreshness());
  }
}
```
